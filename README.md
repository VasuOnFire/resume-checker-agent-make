# Resume Checker Agent (Make.com)

## Overview
Resume Checker Agent is an automation project built using Make.com.
It analyzes resume content and sends notifications through Email and Telegram.

This project demonstrates no-code automation, workflow orchestration,
and real-time messaging using Make.com.

---

## Features
- Automatic resume content extraction
- Resume processing and analysis
- Email notification system
- Telegram message alerts
- Event-based automation using webhooks

---

## Scenarios Included
- Get Resume Content
- Send Email
- Send Message
- Telegram Watch Updates

---

## How to Import in Make.com
1. Open Make.com
2. Create a new scenario
3. Click **Import Blueprint**
4. Upload the `.blueprint.json` file
5. Reconnect required apps (Email, Telegram, Webhooks)
6. Run the scenario
